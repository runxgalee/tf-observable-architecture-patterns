## Task

Launch the cicd-specialist agent to help with CI/CD implementation or bug investigation.

The cicd-specialist agent specializes in:
- GitHub Actions workflows
- CI/CD pipeline design and debugging
- Terraform CI/CD patterns
- Test automation
- Build and deployment issues

### Usage

```
/cicd-specialist [optional: specific task description]
```

If no specific task is provided, the agent will analyze the current CI/CD setup and offer assistance.

### Examples

- `/cicd-specialist` - General CI/CD assistance
- `/cicd-specialist debug failing workflow` - Investigate workflow failures
- `/cicd-specialist add security scanning` - Implement security checks
- `/cicd-specialist optimize terraform plan` - Improve Terraform CI performance
